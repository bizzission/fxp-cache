The Sonatra Cache adds extra features to [Symfony PSR-6 Cache Component](https://github.com/symfony/cache).

Features include:

- Delete only all items by prefix
- Cache adapters:
  - Apcu
  - Array
  - Chain
  - Filesystem
  - Null
  - PHP Array
  - PHP Files
  - Redis
