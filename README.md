The Fxp Cache adds extra features to [Symfony PSR-6 Cache Component](https://github.com/symfony/cache).

Features include:

- Delete only all items by prefix
- Cache adapters:
  - Apcu
  - Array
  - Chain
  - Doctrine
  - Filesystem
  - Memcached
  - Null
  - PDO
  - PHP Array
  - PHP Files
  - Redis
  - Simple Cache
  - Tag Aware
  - Proxy
  - Traceable
  - Traceable Tag Aware
